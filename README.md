# Student Result Processing System

## Description
This project is a Student Result Processing System developed using C language.
The program reads student details from a file, validates the data, calculates
total marks, percentage, grade, and CGPA, and displays individual and class results.

The application is implemented using a modular approach with user-defined
header files.

---

## Project Structure

StudentResultSystem/
├── include/        (Header files)
├── src/            (Source files)
├── data/           (Input file)
├── docs/           (Documentation)
├── main.c
├── README.md
└── .gitignore

---

## Features
- File-based input
- Input validation
- Grade and CGPA calculation
- Class statistics
- Modular design

---

## How to Run
1. Place input file in `data/students.txt`
2. Compile all `.c` files
3. Run the executable
